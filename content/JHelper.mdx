JHelper는 JEdutools의 도움말 페이지를 제작하고, 배포하기 위한 시스템입니다.  
지금 보고 계신 도움말 페이지는 [GitHub 저장소](https://github.com/JBNU-JEduTools/document)의 문서들을 토대로  
자동으로 빌드, 배포한 웹페이지 입니다.  

쉽고 단순하게 설계하여, 아직 SW 개발이 익숙하지 않은 학생들도 쉽게 접근할 수 있습니다.

* Markdown 언어를 사용하여 손쉽게 수정할 수 있습니다.
* GitHub action을 이용한 자동 빌드 시스템을 구축하여 빠르고 쉽게 배포합니다.


## Contribution

Contribution은 [개발자 매뉴얼](https://jhelper.jbnu.ac.kr/JHelper/2devManual)을 참고바랍니다.


## 버그 제보 및 기능 제안

기능 제안은 Repository 내의 Issue 기능을 통해 제안해 주세요 - [바로가기](https://github.com/JBNU-JEduTools/JHelper/issues)

## 원본 프로젝트

JEdutools 문서화 프로젝트는 codebushi의 gatsby-theme-document-example를 수정해 사용하고 있습니다.
```
git clone https://github.com/codebushi/gatsby-theme-document-example
```
