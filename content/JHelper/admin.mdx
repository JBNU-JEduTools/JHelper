# JHelper 관리자 매뉴얼

이 문서는 JHelper 프로젝트의 기술적인 운영 및 관리를 담당하는 관리자를 위한 상세 매뉴얼입니다.
아래 목차를 통해 각 주제별 가이드라인을 확인할 수 있습니다.

---

## 목차

- **[기술 유지보수 가이드](/JHelper/4admin/1architecture)**
  - 프로젝트의 아키텍처, 주요 기술 스택, 핵심 유지보수 작업 및 고급 주제를 다룹니다.

- **[배포](/JHelper/4admin/2deployment)**
  - 프로젝트의 빌드 및 배포 프로세스에 대한 상세 가이드입니다.

- **[레포지토리 관리](/JHelper/4admin/3management)**
  - GitHub 레포지토리의 관리자 추가, Pull Request 검토 등 운영 작업을 안내합니다.

- **[Troubleshooting](/JHelper/4admin/4troubleshooting)**
  - 프로젝트 운영 중 발생할 수 있는 일반적인 문제와 해결 방법을 제공합니다.
