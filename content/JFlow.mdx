## SaaS로 제공하는 오픈소스 DevOps CI/CD 플랫폼

[JFlow](https://jflow.jbnu.ac.kr/)는 실제 배포 프로세스에서 활용되는 여러 오픈소스들을 별도의 설치 없이 즉시 사용가능한 형태로 제공하는 플랫폼입니다.<br/>
교육 환경부터 실무 서비스 개발, 연구 프로젝트에 등 다양한 목적에 맞게 활용할 수 있으며, 인프라 구성 없이 핵심 업무에 집중할 수 있습니다.

## 제공중인 서비스

**GitLab** | https://gitlab.jbnu.ac.kr

**`소스 코드 관리, 이슈 추적` 등 `프로젝트 라이프사이클을 관리하는 오픈소스 통합 개발 플랫폼`입니다.** <br/>
**`코드 협업`과 `프로젝트 관리`를 원활하게 수행할 수 있습니다.**

<br/>

**Jenkins** | https://jenkins.jbnu.ac.kr

**`지속적 통합 및 배포(CI/CD)를 자동화`하는 오픈소스 도구입니다.** <br/>
**코드 변경사항을 자동으로 빌드, 테스트하고 `배포 파이프라인을 구축하여 개발 프로세스를 효율화`합니다.**

<br/>

**Harbor** | https://harbor.jbnu.ac.kr

**`컨테이너 이미지를 안전하게 저장, 관리`할 수 있는 오픈소스 레지스트리입니다.** <br/>
**`취약점 스캐닝과 접근 제어 기능`을 통해 `컨테이너 이미지의 보안을 강화`합니다.**

<br/>

**Sonarqube** | https://sonarqube.jbnu.ac.kr

**`코드 품질과 보안 취약점을 자동으로 검사`하는 `정적 코드 분석 오픈소스 도구`입니다.** <br/>
**`지속적인 코드 품질 모니터링`을 통해 더 안정적이고 유지보수가 용이한 코드를 작성할 수 있습니다.**

<br/>

**Grafana** | https://grafana.jbnu.ac.kr

**다양한 데이터 소스로부터 수집된 정보를 시각화하는 오픈소스 `분석 및 모니터링 플랫폼`입니다.** <br/>
**직관적인 대시보드를 통해 `시스템 성능과 애플리케이션 상태를 실시간으로 모니터링`할 수 있습니다.**
