JEduTools는 전북대학교의 SW 교육을 위한 도구 모음입니다.  

| 이름   | 소개   | 링크   | 도움말 | GitHub 저장소
|---------|---------|:---------:|:---------:|:---------:|
| JCloud  | 전북대학교 클라우드 시스템  | [Click](http://jcloud.jbnu.ac.kr)   |[Manual](https://jcloud-devops.github.io/) | [repo](https://github.com/JBNU-JEduTools/JCloud)   |
| Litmus   | 프로그래밍 과제 자동 채점 시스템   | [Click](http://litmus.jbnu.ac.kr)   |  |  |
| JCode   | Web 기반 IDE 서비스   |  | [Manual](https://jedutools.jbnu.ac.kr/JCode) | [repo](https://github.com/JBNU-JEduTools/JCode) |
| JStackOverflow   | SW 관련 QnA 게시판   | [Click](http://idea.jbnu.ac.kr)   | [Manual](https://jedutools.jbnu.ac.kr/JStackOverflow) | [repo](https://github.com/JBNU-JEduTools/JStackoverflow)   |
| JHelper   | JEduTools 도움말 작성 및 배포 시스템   | [Click](https://jedutools.jbnu.ac.kr/J-Helper)   | [Manual](https://jedutools.jbnu.ac.kr/J-Helper) | [repo](https://github.com/JBNU-JEduTools/JHelper)  |
| Watcher | JCode 기반 코딩 액티비티 수집 및 분석 시스템   |    |   | [repo](https://github.com/JBNU-JEduTools/Watcher)|

* JEduTools는 SW중심대학사업단의 지원으로 컴퓨터인공지능학부에서 운영하고 있습니다.
* JCloud는 정보혁신처 (가칭)클라우드관리팀에서 운영하고 있습니다.

담당: 박현찬 교수 (hyunchan.park@jbnu.ac.kr)

## 모두가 참여하는 오픈소스 프로젝트
JEduTools는 누구나 개발에 참여할 수 있습니다.  
특히, 학생들이 **수업에서 실제 사용하는 도구에 대해 직접 개발에 참여**하도록 하여,
* SW 학습에 대한 동기 부여를 높이고,
* 보다 실전적인 과제로 실무 역량을 높이고,
* 오픈소스 SW 에 대한 장벽을 낮춥니다.

## 자체 구축 클라우드 인프라 기반
자체 구축 및 운영 중인 JCloud 를 기반으로 하여,  
* 구성원들의 요구 사항을 100% 수용할 수 있고,
* 비용 효율적입니다.
